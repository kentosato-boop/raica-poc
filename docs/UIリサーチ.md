# UIリサーチ（2026-07-19）

## 採用した方向

- Appleのシステムフォント、情報階層、控えめなマテリアル、明確な選択状態を参考にする。
- 業務アプリのため、装飾よりも表・リスト・詳細ペイン・操作フィードバックを優先する。
- ライト/ダーク、Reduced Motion、狭幅レイアウトを同一コンポーネントで扱う。
- アイコンはWeb再配布可能なLucideを使用する。
